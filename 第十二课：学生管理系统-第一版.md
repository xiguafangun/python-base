# 学生管理系统-第一版

## 需求概述

新的一个学期开始了，开学的第一天辅导员找到你，说班里学生太多，相让你帮着做一个学生管理系统。
需要能对学生的个人信息完成增删改查的操作。

```python
data = []

while True:
    inp = input('输入你要执行的操作，增加、删除、修改、查找或退出：\n')
    if inp == '增加':
        inp = input('输入姓名、性别、年龄，用空格作区分：\n')
        name, gender, age = inp.split(' ')
        data.append(
            dict(
                name=name,
                gender=gender,
                age=age,
            )
        )
        print('增加成功')
    elif inp == '删除':
        inp = input('输入要删除人员的姓名：\n')
        data = [item for item in data if item['name'] != inp]
        print('删除成功')
    elif inp == '修改':
        inp = input('输入要修改人员的姓名：\n')
        name, gender, age = input('输入姓名、性别、年龄，用空格作区分：\n').split(' ')
        for item in data:
            if item['name'] == inp:
                item['name'] = name
                item['gender'] = gender
                item['age'] = age
    elif inp == '查找':
        inp = input('请输入要查找的人员，输入等于列出所有人员：\n')
        for item in data:
            if inp in item['name']:
                print(
                    '姓名：{name}，性别：{gender}，年龄：{age}'.format(
                        name=item['name'], gender=item['gender'], age=item['age']
                    )
                )
    elif inp == '退出':
        break

    print('\n')

```
